# BarnTrack - Feed Cost Management

This is the BarnTrack application for managing feed costs across multiple farms.

## Development

```bash
npm install
npm run dev -- --port 3004
```

## Build

```bash
npm run build
```

## Tech Stack

- Next.js 15
- Supabase
- TypeScript
- TailwindCSS v4
- shadcn/ui
